#  A/B Testing Case Study

##  Problem Statement

A company wants to evaluate whether a new UI design improves conversion rates compared to the existing design.

---

##  Objective

* Compare Old UI vs New UI
* Measure conversion rate
* Validate results using statistical testing

---

##  Tools Used

* Python (Pandas, Statsmodels, Matplotlib)

---

##  Hypothesis

* **H0:** No difference in conversion rate
* **H1:** New UI improves conversion rate

---

##  Conversion Analysis

![Conversion Table]https://github.com/NuraMishra/AB_Testing/blob/main/conversion_table.png

* Old UI: ~24% conversion
* New UI: ~27% conversion

---

##  Visualization

![A/B Test Chart]https://github.com/NuraMishra/AB_Testing/blob/main/AB_chart.png

The chart clearly shows an increase in conversion rate for the new UI.

---

##  Statistical Test

![Z Test Result]https://github.com/NuraMishra/AB_Testing/blob/main/z_test.png

* Z-statistic: -3.65
* P-value: 0.00025

---

##  Key Insight

The new UI shows a **statistically significant improvement (p < 0.05)**, confirming better performance.

---

##  Business Recommendation

* Roll out the new UI
* Continue A/B testing
* Monitor long-term performance
